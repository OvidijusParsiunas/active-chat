# Full Screen

Examples to help you expand the chat component dimensions.

### Default {#default}

This example uses the default Deep Chat theme.

import ComponentContainer from '@site/src/components/chat/componentContainer';
import DeepChatBrowser from '@site/src/components/chat/deepChatBrowser';
import CodeToggle from '@site/src/components/toggle/codeToggle';
import LineBreak from '@site/src/components/markdown/lineBreak';
import BrowserOnly from '@docusaurus/BrowserOnly';
import TabItem from '@theme/TabItem';
import Tabs from '@theme/Tabs';

<BrowserOnly>{() => require('@site/src/components/nav/autoNavToggle').readdAutoNavShadowToggle()}</BrowserOnly>

<ComponentContainer>
  <DeepChatBrowser
    demo={true}
    history={[
      {text: 'Hey, how are you?', role: 'user'},
      {text: 'I am doing great, thanks.', role: 'ai'},
      {text: 'What is the meaning of life?', role: 'user'},
      {text: 'Seeking fulfillment and personal growth.', role: 'ai'},
    ]}
    style={{
      borderRadius: '10px',
      width: '96vw',
      height: 'calc(100vh - 70px)',
      paddingTop: '10px',
    }}
    messageStyles={{
      default: {
        shared: {
          innerContainer: {
            fontSize: '1rem',
          },
        },
      },
    }}
    inputAreaStyle={{
      fontSize: '1rem',
    }}
    stream={true}
  ></DeepChatBrowser>
</ComponentContainer>


### Simple {#simple}

If you want to automatically scale the inner chat's elements relative to the font-size, you can simply add the desired _font-size_
inside the [`style`](/docs/styles#style) property.

<ComponentContainer>
  <DeepChatBrowser
    demo={true}
    history={[
      {text: 'Hey, how are you?', role: 'user'},
      {text: 'I am doing great, thanks.', role: 'ai'},
      {text: 'What is the meaning of life?', role: 'user'},
      {text: 'Seeking fulfillment and personal growth.', role: 'ai'},
    ]}
    style={{
      borderRadius: '10px',
      width: '96vw',
      height: 'calc(100vh - 70px)',
      fontSize: '1.37rem',
      paddingTop: '10px',
    }}
    stream={true}
  ></DeepChatBrowser>
</ComponentContainer>
