<b>Active Chat</b> is a fully customizable chat component that can be injected into your website with minimal to no effort. It is a close sibling to the [Deep Chat](https://deepchat.dev/) component that has all of its great benefits without the AI related features, making it more lean and quicker to load.

### :rocket: Main Features

- Connect to any API
- Avatars
- Names
- Send/Receive files
- Capture photos via webcam
- Record audio via microphone
- Speech To Text for message input
- Text To Speech to hear message responses
- Support for MarkDown and custom elements to help structure text and render code
- Introduction panel and dynamic modals to help describe functionality for your users
- Connect to popular AI APIs such as OpenAI, HuggingFace, Cohere directly from the browser
- Support for all major ui frameworks/libraries
- Host a model on the browser
- Everything is customizable!
